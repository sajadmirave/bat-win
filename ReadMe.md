# Bat Win
open source cli tools

# Installation

```bash
git clone https://github.com/sajadmirave/bat-win.git
cd bat-win
```

```bash
python setup.py
```

# Commands

### show file list
```bash
bat-ls
```

output like this:

```bash
.git          docs          info.txt      ReadMe.md    
setup.py      src
```

### read file
```bash
bat-cat filename --color =color_name
```

colors:

| color|
|---|
| red  |
| yellow  |
| green  |
| magenta  |
| cyan  |
| blue  |

output like this:

```bash
Hello World
```
